### 算法策略
#### 贪心算法
- 最优装载问题 - C容积的船尽可能装数量足够多的货物
- 背包问题 - 宝物可以分割将，容量有限，宝物的体积和价值 ==》 性价比
- 会议安排 - 在有限多的时间内尽可能多的开会议

- 最短路径
	1. 求源点到所有顶点的所有最短路径
	2. 应用场景：旅行 
  
- 哈夫曼编码
	1. 应用场景：压缩高频数据
	2. 如何避免出现前缀相同编码？

- 最小生成树 
	1. 在图中深度或广度搜索有没有回路是一件繁重的工作 - 闭圈法。
	2. 应用场景：校园网布线
	3. 名词
	```
	# 子图：从原图中选中一些顶点和边组成的图
	# 生成子图：选中一些边和所有顶点组成的图称为原图的生成子图
	# 生成树：生成子图恰好是一颗树
	# 最小生成树：权值之和最小的生成树，称为最小生成树
	```
