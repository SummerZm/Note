### 算法策略
#### 贪心算法
- 最优装载问题 - C容积的船尽可能装数量足够多的货物
- 背包问题 - 宝物可以分割将，容量有限，宝物的体积和价值 ==》 性价比
- 会议安排 - 在有限多的时间内尽可能多的开会议

- 最短路径
	1. 求源点到所有顶点的所有最短路径
	2. 应用场景：旅行 
  
- 哈夫曼编码
	1. 应用场景：压缩高频数据
	2. 如何避免出现前缀相同编码？

- 最小生成树 
	1. 在图中深度或广度搜索有没有回路是一件繁重的工作 - 闭圈法。
	2. 应用场景：校园网布线
	3. 名词
	```
	# 子图：从原图中选中一些顶点和边组成的图
	# 生成子图：选中一些边和所有顶点组成的图称为原图的生成子图
	# 生成树：生成子图恰好是一颗树
	# 最小生成树：权值之和最小的生成树，称为最小生成树
	```

####回溯算法
> 与随机算法紧密相关【源自数学女孩】
- 大卖场购物车
- 部落护卫队
- 地图调色板
- n皇后问题
- 机器零件加工
- 旅行商问题
- 思路
	```
	1. 定义合适的解空间
		a. 解的组织形式：n元组
		b. 显约束：对解分量的取值范围的限定，可以控制解空间的大小
	2. 确定解空间的组织结构: 子集树/排列树/m叉树
	3. 搜索解空间:
		a. 深度优先搜索,不满足条件时回溯
		b. 只求可行解：只需设定约束函数
		c. 求最优解：设定约束函数和界限函数
	4. 回溯法解题的关键是：设计有效的显约束和隐约束
	```
