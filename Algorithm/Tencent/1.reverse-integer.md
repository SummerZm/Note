## **翻转整数的数学论证**

### **数学不等式**
1. −2^31 ≤ (rev⋅10+digit) ≤ 2^31−1

### **相关代码**
```C++
class Solution {
public:
    int reverse(int x) {
        int rev = 0;
        while (x != 0) {
            if (rev < INT_MIN / 10 || rev > INT_MAX / 10) {
                return 0;
            }
            int digit = x % 10;
            x /= 10;
            rev = rev * 10 + digit;
        }
        return rev;
    }
};
```
