## atoi 思维分析

### 2021-07-21
- 字符串解析问题 - 至少要过一遍 O(n)
- 考查空间存储方面的问题: 上下界溢出问题
- 三大基础程序逻辑熟练度：顺序/分支/循环
- 验证：取一个int类型的变量，赋值INT_MAX， 再+1 会是什么结果？
	```C
	/* rollback */
	int test1() {
    	int val = INT_MIN-1;
    	return val;
	}
	// result: 2147483647


	int test2() {
    	int val = INT_MIN-1;
    	return val;
	}
	// result: -2147483648

	/* crash boom */
	int test3() {
    	int val = INT_MAX;
    	return val+1;
	}
	```
